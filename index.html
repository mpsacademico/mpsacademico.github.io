XPROMO: explore seu mundo de interesses

Desenvolvimento de rede social para compras coletivas. O software foi construído aplicando-se conceitos de Estruturas de Dados e Metodologia Ágil. XPROMO é uma espécie de rede social com objetivo de facilitar relações para compras coletivas. Este projeto possui alguns diferenciais como: um interessante trabalho na criação de um modelo de negócios e o programa em si. O programa foi desenvolvido priorizando-se o conceito de estruturas de dados, nesse sentido, ele não possui um banco de dados, todas as informações são mantidas na memória aleatório do computador durante sua execução. Essa foi uma excelente maneira de se aprender a usar as estruturas de dados, sendo elas: vetores, listas, filas, pilhas, árvores e grafos.
